
Escalada de Privilegios con binarios SUID
Buscamos binarios con permisos SUID (pueden ejecutarse como root) .

https://gtfobins.github.io/

```
find / -perm -4000 -ls -user root 2>/dev/null

find / -perm -4000 -ls -user root 2>/dev/null | grep -v snap

find / -perm -4000 -ls >/dev/null | grep -v snap

Binarios que son utiles para elevar privilegios

```
