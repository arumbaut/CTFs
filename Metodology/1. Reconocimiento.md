Reconociendo la red


```
arp-scan -I wlan0 --localnet --ignoredups

nmap -sn 192.168.1.1/24
```

Escaneo primario 

```
nmap -p- --open -sS --min-rate 5000 -vvv -n -Pn 192.168.1.52 -oG ./nmap/allPorts
```

Escaneo mas exaustivo y especifico
```
nmap -p 22,80 -sCV  192.168.1.52 -oN nmap/Target
```

Detectar tecnologias web con whatweb o wapalizer
```
whatweb http://192.168.1.52
```

Podemos recopilar ademas informacion de lo obtenido en nmap poniendo en el buscador seguido de la palabra launchpad, puede der info del SO
```
OpenSSH 7.6p1 Ubuntu 4ubuntu0.3 launchpad

Apache httpd 2.4.29 launchpad
```

Escaneo con scripts de nmap al puerto 80

```
nmap --script http-enum -p80 192.168.1.52 -oN nmap/webScan
```

Directori Traversal con gobuster
```
gobuster dir -u http://192.168.1.52 -w /usr/share/seclists/Discovery/Web-Content/directory-list-2.3-medium.txt -t 20

-t 20   #20 hilos de conexion

gobuster dir -u http://192.168.1.52/blog/archives/ -w /usr/share/dirbuster/wordlists/directory-list-2.3-medium.txt -x php,html,txt -t 30

-x   #Para busqueda de extensiones
```
